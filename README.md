# XIMAGO Outlet — Web oficial

Web del comercio **XIMAGO OUTLET**, con tiendas físicas en **Linares y Andújar (Jaén)**. Proyecto desarrollado para un cliente real, en producción.

🔗 **Ver en vivo:** https://ximago.web.app

## Stack

- JavaScript, HTML5, CSS3
- Firebase Hosting

## Funcionalidades

- Localización, direcciones y horarios de las dos tiendas
- Contacto directo por WhatsApp
- Diseño responsive (móvil y escritorio)
